# ♻️ Campus Cycle

## 🚀 Overview

Campus Cycle is a web-based platform designed for students to **buy, sell, exchange, or donate used items within a campus**. The goal is to promote sustainability, reduce waste, and create a convenient marketplace exclusively for students.

---

## 🎯 Problem Statement

Students often have unused items like books, electronics, or accessories but lack a proper platform to sell or exchange them within their campus.

👉 Campus Cycle solves this by providing a **centralized student marketplace**.

---

## 💡 Key Features

### 👤 User Features

* User Registration and Login
* Add product listings (sell / donate / exchange)
* Browse available items posted by other students
* View product details
* Simple and user-friendly interface

### 🛍 Marketplace System

* Categorized product listings
* Easy navigation and filtering
* Exchange and donation options (not just selling)
